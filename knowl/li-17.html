Item 1.

Open the following url in another tab: https://runestone.academy/runestone/default/user/register 1 . The form is pretty self-explanatory, except for the Course Name field. There is information about that on the right side. You can specify one of the strings that identify one of the public courses that the server knows about: thinkcspy, pythonds, etc. Or you can enter the string for a course that someone else has created and provided you with.
